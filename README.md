# tableau-dashboard-analysis
Interactive Tableau dashboard published on Tableau Public. This dashboard shows the change in fuel prices over time in European Union and Eurozone countries, according to fuel type and tax status.

## Tools Used
- Tableau Desktop
- Tableau Public
- Excel (data source)
- Tableau Extract (.hyper)
